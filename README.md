# Capstone_1
## EmporiUm Sales Analysis
### This project explores sales data for EmporiUm, a student‑focused bookstore chain offering books, tech products, and art supplies through in‑store and online channels. Using SQL for data analysis, I answered sales management questions by region and territory, exported query results to CSV, and created a Pivot Chart in Excel to support insights shared in a demo and live presentation. The sales territory I have is South Carolina.
